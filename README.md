# Introduction to COBOL Programs
This repository contains a collection of introductionary COBOL programs.

## Build & Run COBOL Programs
To compile a COBOL program (hello.cob), use ```.\cobbuild.bat -x .\01_HelloWorld\hello.cob -o .\01_HelloWorld\hello.exe -lcob```

To run the compiled program, use ```.\01_HelloWorld\hello.exe```
